Motion Detection Security System (Arduino + Ultrasonic Sensor + Buzzer)

This project implements a simple motion-detection security system designed for safeguarding valuables such as jewellery. <br> The system uses an HC-SR04 ultrasonic sensor to detect movement within a specific range. When motion is detected, a buzzer and LED are triggered to alert the user.
